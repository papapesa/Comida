Block Blast

Block Blast é um jogo de quebra-cabeça viciante onde o objetivo é preencher linhas ou colunas inteiras com blocos coloridos. Diferente do Tetris tradicional, o jogador posiciona as peças livremente até não haver mais espaço disponível.

Mecânica

O jogo apresenta peças de formatos variados que devem ser arrastadas para a grade 10x10. Quando uma linha ou coluna é preenchida, ela é eliminada e o jogador marca pontos.

Desenvolvimento

Block Blast foi desenvolvido por estúdios independentes e rapidamente se popularizou por sua simplicidade e desafio progressivo.
